Project Overview

This project is a personal e-profile / portfolio website created to showcase my academic background, technical skills, projects, hobbies, certifications, and contact information in a professional format. The website is designed using HTML and CSS and deployed online using GitHub Pages so that it can be accessed from anywhere through a live link.

The purpose of this project is to build an online professional profile that represents my skills, achievements, and interests as an engineering student.

Steps to Run the Project Locally

1. Download or clone the repository:
   
2. Open the project folder.
3. Open the "index.html" file in any web browser.

Steps to Deploy on GitHub Pages

1. Create a GitHub repository.
2. Upload the project files ("index.html", "style.css", and README).
3. Go to Repository Settings.
4. Open Pages section.
5. Under Source, choose Deploy from branch.
6. Select main branch and /root folder.
7. Save the settings.
8. GitHub will generate a live website link.
